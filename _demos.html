Form with CSS Error Messages

In this demo the error messages will be shown thanks to CSS classes. All error messages are already insdie the HTML.

Write your name!
Email address cannot be empty!
Check your email address, it is not valid:
  • @ char is missing
  • Username is missing
  • Domain is missing
  • Extension dot is missing
  • Extension is missing
  • Extension isn't long enough
Username cannot be empty!
Username is not valid:
  • Must be al least 3 chars long
  • Must be at most 24 chars long
  • Contains not allowed chars. It must be with letters, numbers, underscores and/or . - @
  • Must start with a letter, number or underscore
Password cannot be empty!
Check your password, it is not valid:
  • Must have least 8 chars
  • Must contain a number
  • Must contain a lowercase char
  • Must contain an uppercase char
  • Can contain only letters and numbers
Password and Confirm Password are not equal!
